## 🔥 实战项目

### 📡 [iot-alert-system](https://github.com/lxt-321/iot-alert-system)
**基于 FastAPI + MySQL + Redis 的物联网设备监控告警系统**

- 设备生命周期管理：JWT 注册鉴权、心跳检测、在线状态追踪
- 规则引擎：动态阈值告警、设备状态机、温度/湿度等传感器数据处理
- 实时推送：WebSocket 双向通信、Redis 冷却去重、告警确认/解决工作流
- 管理后台：设备 Dashboard、告警统计、可视化监控面板
- 技术栈：Python 3.11 · FastAPI · SQLAlchemy 2.0 · MySQL 8.0 · WebSocket · JWT

> 25 个文件、3000+ 行代码，含完整 Docker 部署、API 文档（Swagger）、端到端测试。
